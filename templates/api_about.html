API

The Bookkase API provides developers with access to Bookkase's database of book information and ratings. To get started using the Bookkase API, please read below.

The Bookkase API currently offers general information and rating data for individual works. This data can be retrived in JSON format via the url path /api/books/[ISBN] where [ISBN] is the ISBN of the work whose data you would like to access.

The API will return the following keys:

"author" = The work's author

"average_score" = The work's average rating (maximum of 5.0) on Bookkase

"isbn" = The work's ISBN

"review_count" = The number of ratings the work has received on Bookkase

"title" = The work's title

"year_published" = The year in which the work was published
